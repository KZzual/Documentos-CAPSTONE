# Repositorio de APIs y Links de Estudio - Informática

## APIs Públicas para Desarrollo

### APIs de Datos Académicos
- **Universities List API**: http://universities.hipolabs.com/search?country=
    - Información de universidades por país

### APIs de Programación y Código
- **GitHub API**: https://api.github.com
    - Gestión de repositorios y proyectos
- **Stack Exchange API**: https://api.stackexchange.com/docs
    - Preguntas y respuestas de programación

### APIs de Aprendizaje
- **Quotes API**: https://api.quotable.io/random
    - Citas motivacionales para estudiantes
- **Dictionary API**: https://api.dictionaryapi.dev/api/v2/entries/en/{word}
    - Definiciones de términos técnicos

### APIs de Utilidad
- **JSONPlaceholder**: https://jsonplaceholder.typicode.com
    - API de prueba para desarrollo
- **Random User API**: https://randomuser.me/api/
    - Datos de usuarios para testing
